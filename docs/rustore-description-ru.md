# Описание приложения для RuStore

## Краткое описание

Будильник, который выключается только после решения заданий.

## Основное описание

Умный будильник помогает проснуться и не отключить сигнал машинально. Чтобы остановить будильник, необходимо выполнить заданную последовательность головоломок.

Возможности приложения:

• математические примеры;
• задания на ввод текста;
• упражнения на память;
• отключение встряхиванием телефона;
• выбор типов и количества заданий;
• однократные и повторяющиеся будильники;
• выбор мелодии и постепенное увеличение громкости;
• дополнительные проверки после пробуждения;
• восстановление расписания после перезагрузки устройства.

Приложение работает без регистрации, рекламы и подключения к интернету. Настройки и данные будильников хранятся только на устройстве и не передаются третьим лицам.
